# AirMic WR104 使用说明（公开版）

**版本**：`wr1.0.4`  
**发布日期**：2026-06-26  
**作者**：**Aiya FUN**

AirMic WR104 通过蓝牙提供两项核心能力：  
1) 蓝牙麦克风输入，2) 蓝牙键盘快捷输入。  
适用于 Mac / Windows 的语音输入与轻量办公场景。

---

## 你可以怎么用
- 基础语音输入：把 AirMic 设为麦克风输入设备  
- 语音输入加快捷键：用右 Option 触发听写  
- 快捷操作：Backspace / Enter 可直接在文本框输出控制键

## 常见场景

### 场景 1：语音输入法（豆包 / 微信 / 系统）

连接 `AirMic WR104` 后，可以在豆包语音输入法中把它作为语音输入麦克风使用。打开需要输入文字的位置，启动豆包语音输入，对着 AirMic WR104 说话即可完成语音转文字。

适合：

- 长文本输入
- 会议内容整理
- 想法快速记录
- 聊天回复
- 文档草稿输入

### 场景 2：文字与聊天输入

连接 `AirMic WR104` 后，可以在微信输入法中使用语音输入。将光标放到聊天框、文档、网页输入框或其他文本区域，启动微信输入法的语音输入功能后，对着 AirMic WR104 说话即可。

适合：

- 微信聊天
- 企业微信回复
- 文档输入
- 表单填写
- 搜索框输入

### 场景 3：第三方语音输入

AirMic WR104 也可以配合 Mac、Windows 或其他第三方语音输入工具使用。只要当前语音输入工具支持选择系统麦克风，就可以选择 `AirMic WR104` 作为输入设备。

常见使用方式：

- Mac 系统语音输入
- Windows 语音输入
- 浏览器里的语音输入
- 会议软件的麦克风输入
- AI 应用里的语音输入
- 其他支持麦克风输入的软件

### 已经在用时

如果 AirMic WR104 已经连接并正在使用，可以直接在不同软件之间切换输入位置。例如从聊天窗口切到文档，从浏览器切到会议软件，只需要确认当前软件使用的麦克风是 `AirMic WR104`。

如果正在输入过程中发现没有声音：

- 先检查当前软件是否有麦克风权限。
- 再检查系统声音输入是否选择了 `AirMic WR104`。
- 如果切换过电脑或蓝牙设备，建议重新连接一次。

如果长时间没有使用，AirMic WR104 可能会进入省电状态。按任意按键唤醒后，等待几秒即可继续使用。

## 蓝牙配对（操作步骤）

首次使用时：

1. 打开电脑或手机的蓝牙设置。
2. 搜索蓝牙设备 `AirMic WR104`。
3. 点击连接或配对。
4. 连接完成后，系统会识别到 AirMic WR104 的键盘和麦克风功能。

如果之前连接过其他电脑：

1. 在旧电脑的蓝牙设置里删除或忽略 `AirMic WR104`。
2. 在新电脑上重新搜索 `AirMic WR104`。
3. 如果搜索不到，请让设备重新进入配对状态后再搜索。

重新开放配对：

1. 同时按住 `Enter` 和 `右 Option` 约 3 秒。
2. 回到电脑蓝牙设置里重新搜索 `AirMic WR104`。

清除旧绑定：

1. 同时按住 `Backspace` 和 `Enter` 约 5 秒。
2. 等设备重启后，重新搜索并连接 `AirMic WR104`。

## Mac 使用

连接蓝牙：

1. 打开 `系统设置`。
2. 进入 `蓝牙`。
3. 找到 `AirMic WR104` 并连接。

选择麦克风：

1. 打开 `系统设置`。
2. 进入 `声音`。
3. 在 `输入` 里选择 `AirMic WR104`。
4. 对着设备说话，观察输入音量是否有变化。

使用语音输入：

1. 打开需要输入文字的 App。
2. 将光标放到文本输入框。
3. 使用系统语音输入或 App 内的语音功能。
4. 如果系统询问麦克风权限，请允许当前 App 使用麦克风。

## Windows 使用

连接蓝牙：

1. 打开 `设置`。
2. 进入 `蓝牙和设备`。
3. 点击 `添加设备`。
4. 选择 `蓝牙`。
5. 找到 `AirMic WR104` 并连接。

选择麦克风：

1. 打开 `设置`。
2. 进入 `系统`。
3. 进入 `声音`。
4. 在 `输入` 里选择 `AirMic WR104`。
5. 对着设备说话，观察输入音量是否有变化。

使用语音输入：

1. 打开需要输入文字的 App。
2. 将光标放到文本输入框。
3. 使用 Windows 语音输入或 App 内的语音功能。
4. 如果系统询问麦克风权限，请允许当前 App 使用麦克风。

## 常见问题（先看这里）

### 搜索不到 AirMic WR104

请确认设备已开机，并让设备重新进入配对状态。可以同时按住 `Enter` 和 `右 Option` 约 3 秒，然后回到蓝牙设置里重新搜索。

### 点连接后一直转圈

请先在电脑蓝牙设置里删除旧的 `AirMic WR104` 记录，然后重新配对。如果仍然不行，请同时按住 `Backspace` 和 `Enter` 约 5 秒清除旧绑定，再重新连接。

### 连接后没有麦克风输入

请到系统声音设置里确认输入设备已经选择 `AirMic WR104`。如果正在使用浏览器或会议软件，请确认该软件已经获得麦克风权限。

### 连接后键盘没有反应

请确认蓝牙设备状态显示为已连接。如果之前配对失败，建议删除 `AirMic WR104` 后重新配对。

### Mac 上看到了设备，但不能正常使用

请在蓝牙设置里删除 `AirMic WR104`，再重新配对。必要时重启 Mac 蓝牙，或者清除 AirMic WR104 的旧绑定后重新连接。

### Windows 上只出现麦克风或只出现键盘

请删除 `AirMic WR104`，重新添加蓝牙设备。Windows 有时会分步骤识别蓝牙键盘和蓝牙音频，重新配对通常可以恢复完整功能。

### 换电脑使用时连不上

AirMic WR104 默认优先连接之前绑定过的设备。换电脑使用前，请先在旧设备上删除 `AirMic WR104`，或者清除 AirMic WR104 的旧绑定后重新配对。

### 设备长时间不用后不见了

AirMic WR104 长时间未连接时会进入省电状态。按任意按键唤醒，等待几秒后再到蓝牙设置里连接。

### 是否需要安装软件或驱动

基础键盘和麦克风功能不需要安装额外软件。Mac 和 Windows 都可以通过系统蓝牙和系统声音输入功能使用。

### 可以修改按键功能吗

可以。AirMic WR104 的按键功能可以根据使用场景定制，例如常用快捷键、确认键、删除键、语音输入相关按键等。

## 使用建议

- 第一次连接时，建议让设备靠近电脑。
- 如果配对异常，优先删除旧蓝牙记录后重新连接。
- 如果要换电脑使用，建议先清除旧绑定。
- 使用语音输入前，请确认当前软件有麦克风权限。
- 如果长时间不用，设备会进入省电状态，按任意按键即可唤醒。
